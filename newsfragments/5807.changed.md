* The boolean `abi3` field of `pyo3_build_config::impl_::InterpreterConfig` is now
  deprecated and has been replaced by a new `target_abi` accessor function, which returns a `PythonAbi`
  struct.
* The InterpreterConfig::from_interpreter function now accepts a second
  `stable_abi_version` argument. The old behavior is the same as passing
  `None`.
